Эта программа нужна для генерации рандомных паролей.
Для установки нужно ввести эти команды в терминале:
apt-get install python3
apt-get install git
git clone https://github.com/ruslanchr/randompass.git
pip3 install random
cd randompass
python3 randpass.py
Готово!
